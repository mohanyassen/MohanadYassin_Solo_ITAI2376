# Smart AI Health Assistant 🤖

## Project Overview
This project is a simple AI Health Assistant that helps users with basic wellness guidance such as sleep, stress, hydration, and nutrition awareness.

The assistant uses a rule-based AI agent structure with tools and memory to simulate intelligent behavior.

---

## Problem It Solves
Many people struggle with maintaining healthy habits like sleep, hydration, and stress management. This assistant provides simple, accessible health guidance.

---

## Features
- Tracks user sleep and weight
- Provides nutrition information for common foods
- Gives general health advice (stress, tiredness, hydration)
- Stores user data using local memory (JSON)
- Uses tool-based reasoning system

---

## How It Works (Architecture)
User Input → AI Agent → Tool Selection → Memory Check → Response

Tools:
- Nutrition Tool (food calorie lookup)
- Health Advice Tool (rules-based suggestions)

Memory:
- Stores user weight and sleep info locally in JSON file

---

## Tools & Technologies
- Python
- Rule-based AI Agent design
- JSON for memory storage
