# Reflection

## What worked well
The rule-based agent system worked well for handling user input and providing structured responses. The memory system successfully stored user data like sleep and weight.

---

## What did not work well
At first, GitHub upload and Git commands were challenging, and the system initially felt too basic.

---

## Biggest Challenge
The biggest challenge was structuring the project into an AI agent format with tools and memory. This was solved by breaking the system into smaller components (agent, tools, memory).

---

## Changes from Midterm Plan
The original plan included complex AI models and APIs, but the final implementation focused on a simplified rule-based agent for reliability and successful completion.

---

## Future Improvements
If continued, I would integrate a real language model (like GPT), expand health data sources, and improve personalization features.
